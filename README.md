# Cottage Manager (Gestionnaire de locations)

![](https://github.com/koeltv/cottage-manager/workflows/tests/badge.svg)
[![Stable release](https://badgen.net/github/release/koeltv/cottage-manager)](https://github.com/koeltv/cottage-manager/releases)
[![GitHub license](https://badgen.net/github/license/koeltv/cottage-manager)](https://github.com/koeltv/cottage-manager/blob/master/LICENSE)

Cette application a pour but de faciliter la gestion de locations 
sur plusieurs gîtes simultanément.

## Installation

Le logiciel est disponible au téléchargement via des installers pour chaque platforme
via l'onglet [Releases](https://github.com/koeltv/cottage-manager/releases).

Ceux-ci installeront directement l'application et ne nécessitent aucune dépendance supplémentaire.
